Tiny rounded pill for status and category labels — e.g. the plum "First try" tag on a task, "Archived", "Current goal".

```jsx
<Badge tone="info">First try</Badge>
<Badge tone="reward" soft>Current goal</Badge>
<Badge tone="neutral" soft>Archived</Badge>
```

Tones: `info` (plum), `reward` (gold), `action` (teal), `danger` (coral), `neutral`. `soft` uses a tinted background with colored text — quieter, good for inline status next to text.
