# Star Board

A reward system PWA for autistic children. Fully local, no network required, no accounts.

## Quick start

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

## Build for production

```bash
npm run build
```

Output goes to `dist/`. Serve it with any static file server.

## Core principles

- **P1** Check capability before rewarding (may not be able, not just won't)
- **P2** Soft exit always — child can decline, loses nothing earned
- **P3** First-try tasks pay full on attempt, not on success
- **P4** Earned is permanent — ledger is append-only, no star removal
- **P5** Fade plans required on every task
- **P6** Child has a voice in goal/reward selection
- **P7** Default to daily-living/first-try
- **P8** Calm by default, accessible, image-first
- **P9** Fully local, no network, no accounts
